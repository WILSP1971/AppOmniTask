Antes de instalar

  1. Permite instalar desde esta fuente. OmniTask no está en Play Store todavía, así que Android va a pedir habilitar "instalar apps desconocidas" para el navegador o gestor de archivos que uses para abrir el .apk. Es un permiso por app, no un ajuste global.
  2. Descarga el archivo con el botón de arriba. Pesa entre 50 y 60 MB según la versión — con datos móviles puede tardar un poco.
  3. Ábrelo desde notificaciones o descargas. Confirma la instalación cuando el sistema te lo pida.
Los recordatorios push todavía no llegan Esta build no tiene un proyecto de Firebase configurado, así que las notificaciones automáticas de WhatsApp/push están inactivas por ahora. El resto — calendario, actividades, contactos, pendientes por programar y notificaciones locales con la app abierta — funciona contra la API real de producción.
